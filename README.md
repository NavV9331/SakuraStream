# SakuraStream 🌸


**SakuraStream** is the ultimate entertainment hub—a premium, beautifully crafted platform merging over 6,000+ free live global TV channels, an ad-free music studio, and on-demand movies and series into one seamless experience.

## ✨ Features

- **📺 Live World TV:** Dive into 6,000+ aggregated free TV channels streaming directly from nations across the globe. No subscriptions required.
- **🎬 On-Demand Movies & Series:** Stream the latest blockbuster movies and binge-worthy TV series on demand in HD quality, featuring a fully responsive custom player (Powered by TMDb).
- **🎵 Premium Music Hub:** Stream thousands of ad-free tracks, dynamic regional playlists, and explore lyrics dynamically.
- **🎮 Smart TV & Console Optimized:** Fully supports 10-foot UI with a custom 2D Spatial Navigation engine. Navigate the entire platform flawlessly using a TV remote D-Pad or keyboard arrow keys!
- **📱 Mobile Ready:** Heavily optimized for mobile devices with large touch targets and performance overrides that disable heavy GPU filters on lower-end devices to guarantee a buttery smooth 60FPS experience.
- **⚡ Smart Sorting:** Locate global media instantly via hyper-fast region and language filtering, all debounced for zero-lag typing.

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/NavV9331/SakuraStream.git
   cd SakuraStream
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:5173` to experience SakuraStream!

## 🛠️ Built With

- **React & Vite** - High performance frontend rendering and build tooling.
- **Vanilla JS** - Custom lightweight VOD routing and spatial navigation.
- **TMDb API** - Providing comprehensive movie and TV metadata.
- **Lucide React** - Beautiful, consistent iconography.
